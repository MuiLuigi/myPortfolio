Mario and Luigi Blog Webpage Project

This project is my latest project for web development. In this project, I implemented everything I have learned so far for web development. In this project, I used HTML to create the basic structure of the webpage, like the buttons that allows the user to create a new blog or update/delete an existing blog. I used CSS to style the webpage and add some hover effects with the buttons that takes the user to the webpage where they could create new blogs or update/delete existing blogs. I also used JavaScript to save the blog in the memory. I used eventListener for the system to know when the user clicks the submit button and show them a message that tells the user their blog has been saved successfully after filling in the required fields in the form. I used the ".push()" function to add every new blog whenever the user creates a new blog and saves it into the memory. The user is able to select which blog they would like to edit or delete in the existing blogs webpage. Another major concept I learned while working on this project is working with GitHub and Git control.
